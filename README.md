# 📊 E-Commerce Sales Dashboard | Power BI

## 📌 Project Overview

This project is an interactive **E-Commerce Sales Dashboard** developed using **Microsoft Power BI** to analyze sales performance across different brands, states, product categories, and order statuses.

The dashboard provides meaningful business insights through dynamic visualizations, helping users understand sales trends, customer purchasing patterns, and overall business performance.

---

## 🚀 Features

- 📈 Total Sales Analysis
- 📦 Total Quantity Sold
- 🛒 Total Orders Overview
- 🏷️ Brand-wise Sales Analysis
- 🌍 State-wise Sales Distribution
- 📅 Quarterly Sales Trend
- 📊 Product Category Analysis
- 📌 Order Status Tracking
- 🎯 Interactive Filters
- 📉 Business KPI Dashboard

---

## 🛠️ Tools & Technologies

## 🎯 Business Objective

The objective of this project is to transform raw sales data into an interactive dashboard that enables business users to:

- Monitor overall sales performance
- Identify top-performing brands
- Analyze regional sales
- Track order status
- Make data-driven business decisions
